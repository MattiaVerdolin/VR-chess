\doxylink{class_persp_camera}{Persp\+Camera} include file. 

This file defines the \doxylink{class_persp_camera}{Persp\+Camera} class, which represents a perspective camera in a 3D scene. A perspective camera simulates a camera that uses perspective projection, where objects appear smaller as they are farther from the camera.
