\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_texture}{Texture} class, a concrete class for handling 2D textures in the rendering engine. 

This file defines the \doxylink{class_texture}{Texture} class, which provides functionality for loading, managing, and rendering 2D textures. It also supports anisotropic filtering if the feature is available on the system.
