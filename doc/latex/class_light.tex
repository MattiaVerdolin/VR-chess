\doxysubsection{Detailed Description}
Abstract base class for light sources. 

The \doxylink{class_light}{Light} class is an abstract base class that represents a light source in a 3D scene. It manages the position, intensity, and on/off status of the light, and allows for rendering the light as part of a scene. Derived classes can provide specific light types and behavior.
