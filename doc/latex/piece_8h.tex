\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_piece}{Piece} class, representing a game piece in a chessboard scene. 

This file defines the \doxylink{class_piece}{Piece} class, which encapsulates a game piece\textquotesingle{}s position (row and column) and its associated node in the 3D scene. The class provides methods for manipulating the piece\textquotesingle{}s position and accessing the associated node.
