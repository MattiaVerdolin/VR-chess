\doxysubsection{Detailed Description}
Singleton class for managing event listeners and notifying them of events. 

The \doxylink{class_notification_service}{Notification\+Service} class provides functionality to subscribe and unsubscribe listeners to window reshape and matrix change events. It ensures that when an event occurs, all subscribed listeners are notified.
