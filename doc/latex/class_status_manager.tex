\doxysubsection{Detailed Description}
Singleton class for managing game states and their associated behaviors. 

The \doxylink{class_status_manager}{Status\+Manager} class maintains a mapping of game states to their callbacks and menus. It also allows for the registration of listeners that are notified when the game state changes.
