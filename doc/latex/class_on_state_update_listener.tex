\doxysubsection{Detailed Description}
Class for handling updates when the game state changes. 

The \doxylink{class_on_state_update_listener}{On\+State\+Update\+Listener} class maps game states to their corresponding handler functions. Derived classes can override the virtual handler methods to provide specific behavior for each game state.
