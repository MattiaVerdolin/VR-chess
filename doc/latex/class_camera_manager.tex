\doxysubsection{Detailed Description}
Singleton class for managing cameras in a 3D rendering scene. 

The \doxylink{class_camera_manager}{Camera\+Manager} handles the creation, initialization, and management of different cameras used in the 3D chessboard game. It provides functionality to add and remove cameras, change the main camera, and move the camera\textquotesingle{}s position. It also listens to game state changes to update the active camera accordingly.
