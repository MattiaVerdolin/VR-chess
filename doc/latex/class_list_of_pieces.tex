\doxysubsection{Detailed Description}
Class representing a collection of \doxylink{class_piece}{Piece} objects. 

The \doxylink{class_list_of_pieces}{List\+Of\+Pieces} class encapsulates a collection of pieces on the chessboard. It provides various methods for interacting with the list, including removing pieces, cloning the list, and retrieving pieces by their index or coordinates.
