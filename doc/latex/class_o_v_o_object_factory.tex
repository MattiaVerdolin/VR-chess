\doxysubsection{Detailed Description}
Factory class to create objects based on chunk ID and data. 

The {\ttfamily \doxylink{class_o_v_o_object_factory}{OVOObject\+Factory}} class provides a static method {\ttfamily \doxylink{class_o_v_o_object_factory_a848f1eef3f7ebe5eac8b257a3aadd23b}{create\+Object\+By\+Chunk\+ID()}} to create various objects (e.\+g., nodes, materials, meshes, lights) depending on the provided chunk ID and data. The factory class ensures that objects are created in a structured manner based on their type and subtype.
