\doxysubsection{Detailed Description}
Concrete \doxylink{class_ortho_camera}{Ortho\+Camera} class. 

The \doxylink{class_ortho_camera}{Ortho\+Camera} class represents a camera that uses an orthographic projection, where objects are projected without perspective distortion (i.\+e., parallel lines remain parallel). It inherits from the \doxylink{class_camera}{Camera} class and implements the necessary methods to load and apply the orthographic projection matrix.
