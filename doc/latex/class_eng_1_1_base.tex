\doxysubsection{Detailed Description}
Main class for the graphics engine. Implements the Singleton pattern. 

The {\ttfamily \doxylink{class_eng_1_1_base}{Base}} class is responsible for initializing the engine, managing scenes, handling user input, and managing frame rate. It is designed as a singleton to ensure only one instance of the engine exists.
