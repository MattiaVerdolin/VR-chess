\doxysubsection{Detailed Description}
Header file for the \doxylink{class_mesh}{Mesh} class. 

This file contains the declaration of the \doxylink{class_mesh}{Mesh} class, which extends the \doxylink{class_node}{Node} class and represents a 3D mesh in the scene. The class includes methods for rendering, cloning, parsing, and retrieving vertex data.
