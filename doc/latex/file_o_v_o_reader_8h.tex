\doxylink{class_file_o_v_o_reader}{File\+OVOReader} include file. 

This file defines the \doxylink{class_file_o_v_o_reader}{File\+OVOReader} class, which is responsible for reading and parsing {\ttfamily .ovo} files, a custom binary format. It handles the file opening, chunk reading, material retrieval, and node parsing. This class uses a singleton pattern to ensure only one instance is used for file processing.
