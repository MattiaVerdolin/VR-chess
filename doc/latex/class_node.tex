\doxysubsection{Detailed Description}
Represents a node in a hierarchical 3D scene graph. 

A \doxylink{class_node}{Node} stores its local transformation matrix, maintains parent-\/child relationships, and can hold a material for rendering. It supports recursive operations for transformations and notifications.
