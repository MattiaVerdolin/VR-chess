\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_status_manager}{Status\+Manager} class for managing game states and callbacks. 

The \doxylink{class_status_manager}{Status\+Manager} class is a singleton responsible for managing game states, their associated callbacks, and state-\/specific menus. It also allows listeners to subscribe to state updates and triggers appropriate callbacks during state transitions.
