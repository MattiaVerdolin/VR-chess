\doxysubsection{Detailed Description}
Concrete \doxylink{class_persp_camera}{Persp\+Camera} class. 

The \doxylink{class_persp_camera}{Persp\+Camera} class represents a perspective camera in a 3D scene. It simulates a camera using perspective projection, where objects appear smaller as they get further from the camera.
