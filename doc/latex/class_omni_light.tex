\doxysubsection{Detailed Description}
Concrete \doxylink{class_omni_light}{Omni\+Light} class. 

The \doxylink{class_omni_light}{Omni\+Light} class represents an omnidirectional light source in a 3D scene. Unlike spotlights, which emit light in a specific direction, an omnidirectional light emits light uniformly in all directions from a single point.
