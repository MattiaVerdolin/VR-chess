\doxysubsection{Detailed Description}
Concrete \doxylink{class_file_o_v_o_reader}{File\+OVOReader} class. 

The \doxylink{class_file_o_v_o_reader}{File\+OVOReader} class is responsible for parsing {\ttfamily .ovo} files. It reads the file, processes the chunks of data, retrieves materials, and constructs a hierarchical scene graph of nodes from the parsed data.
