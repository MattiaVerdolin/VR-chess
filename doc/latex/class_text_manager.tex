\doxysubsection{Detailed Description}
Class for managing text rendering. 

The {\ttfamily \doxylink{class_text_manager}{Text\+Manager}} class handles rendering text on the graphical window using an orthographic {\ttfamily \doxylink{class_camera}{Camera}} for projection. It supports rendering a list of strings and displaying frames per second (FPS).
