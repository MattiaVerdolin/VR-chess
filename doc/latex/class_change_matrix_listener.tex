\doxysubsection{Detailed Description}
Interface for receiving notifications when a matrix changes. 

The \doxylink{class_change_matrix_listener}{Change\+Matrix\+Listener} interface defines a method that is called whenever a matrix is changed. Classes that want to receive these notifications must implement the on\+Matrix\+Change method.
