\doxysubsection{Detailed Description}
Class representing a game piece with its position and associated node in the scene. 

The \doxylink{class_piece}{Piece} class encapsulates the position of a game piece (row and column) and maintains a reference to the associated \doxylink{class_node}{Node} that represents the piece\textquotesingle{}s visual representation in the 3D scene. It allows modification of the position and provides access to the \doxylink{class_node}{Node} object.
