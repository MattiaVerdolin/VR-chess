\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_light_manager}{Light\+Manager} class, responsible for managing light states in the 3D scene. 

This file defines the \doxylink{class_light_manager}{Light\+Manager} class, which handles the activation, deactivation, and intensity adjustments of lights in the scene. It ensures the proper interaction with the light objects and follows the Singleton design pattern to ensure a single instance.
