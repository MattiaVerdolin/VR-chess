\doxysubsection{Detailed Description}
Interface for receiving notifications when a window is resized. 

The \doxylink{class_reshape_listener}{Reshape\+Listener} interface defines a method that is called whenever the window size changes. Classes that want to be notified of window reshape events must implement the on\+Window\+Reshape method.
