Declaration and implementation of the \doxylink{class_movement_manager}{Movement\+Manager} class, handling movement logic for game pieces. 

This file defines the \doxylink{class_movement_manager}{Movement\+Manager} class, responsible for managing the movement of pieces on the game board. The class includes functionality for moving pieces in all four directions (left, right, up, down), as well as managing turns between two players (white and black).
