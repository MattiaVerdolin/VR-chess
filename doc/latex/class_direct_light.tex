\doxysubsection{Detailed Description}
Concrete \doxylink{class_direct_light}{Direct\+Light} class. 

The \doxylink{class_direct_light}{Direct\+Light} class represents a directional light source in a 3D scene. Unlike other light types, a directional light has no position but instead shines with parallel rays in a specific direction. The \doxylink{class_direct_light}{Direct\+Light} class extends the abstract \doxylink{class_light}{Light} class, providing specific behavior for a directional light.
