\doxysubsection{Detailed Description}
Class for managing the movement of pieces on the board. 

The \doxylink{class_movement_manager}{Movement\+Manager} class handles the logic for moving pieces in four directions (left, right, up, down) and managing the turn-\/based system. It updates the game state and transforms the position of pieces based on player turns.
