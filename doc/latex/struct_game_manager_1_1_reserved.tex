\doxysubsection{Detailed Description}
Struct to hold references to various game subsystems. 

This struct holds references to game components such as camera, status manager, movement manager, list of pieces manager, history manager, light manager, engine, and root nodes for the game scene. It is used internally by the \doxylink{class_game_manager}{Game\+Manager} class to interact with these subsystems.
