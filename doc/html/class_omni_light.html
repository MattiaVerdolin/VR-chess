Detailed Description

Concrete OmniLight class.

The OmniLight class represents an omnidirectional light source in a 3D scene. Unlike spotlights, which emit light in a specific direction, an omnidirectional light emits light uniformly in all directions from a single point.

Constructor for initializing the omnidirectional light with specific parameters.

This constructor creates an OmniLight object with the given name and position. The cutoff angle is set to the default of 180 degrees, as it emits light in all directions.

Parameters
nameThe name of the omni light (default: "defaultOmniLight").
positionThe position of the omni light in 3D space (default: glm::vec3(0.0f)).
