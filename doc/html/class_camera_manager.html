Detailed Description

Singleton class for managing cameras in a 3D rendering scene.

The CameraManager handles the creation, initialization, and management of different cameras used in the 3D chessboard game. It provides functionality to add and remove cameras, change the main camera, and move the camera's position. It also listens to game state changes to update the active camera accordingly.
