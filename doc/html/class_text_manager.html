Detailed Description

Class for managing text rendering.

The TextManager class handles rendering text on the graphical window using an orthographic Camera for projection. It supports rendering a list of strings and displaying frames per second (FPS).

Renders a list of text strings on the window.

This method uses orthographic projection to render text on the graphical window. Each string is displayed starting at the defined vertical position, incrementing the vertical position for each line of text.
