Detailed Description

Factory class to create objects based on chunk ID and data.

The OVOObjectFactory class provides a static method createObjectByChunkID() to create various objects (e.g., nodes, materials, meshes, lights) depending on the provided chunk ID and data. The factory class ensures that objects are created in a structured manner based on their type and subtype.
